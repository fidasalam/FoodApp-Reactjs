# FoodOrder App

FoodOrder is a web application built with React for ordering food online.

## Features

- View menu items
- Add items to the cart
- Remove items from the cart
- View offers
- Learn about the restaurant
- Place orders

## Technologies Used

- React
- JavaScript
- HTML
- CSS
- Bootstrap
